# upload_diff_progress

差分書込の実装と確認の進捗を管理する正本です。  
`upload_diff_plan.md` は旧版の参照用として扱い、以後の更新はこのファイルのみで行います。

## 現在地

- 差分対象は **授業の教室配当のみ**
- 教室マスタは差分対象外
- `書込` は **最新クラウドに対する差分反映**
- `取得` は **クラウド内容のローカル反映**
- コンフリクト時は **B を正**
- 差分内容は **CSV で確認可能**
- `upload_diff_plan.md` は参照用
